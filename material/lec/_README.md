# Class 13 - Gateways, Load Balancers, and Observability

To generate excalidraw diagrams to png files:

```bash
npx @moona3k/excalidraw-export diagram.excalidraw scenario-01-architecture.excalidraw -o output.png --no-background --scale 3
```

Then to resize to fit on slides:

```bash
magick input.png -resize 70% output.png
```
